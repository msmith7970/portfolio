Swiss Tournament Page

In this project I built a python program to simulate the results of a Swiss Pairing Style of tournament. In this style of tournament players in a tournament are not eliminated when they lose a match, instead they are paired in each round with opponents having the same win/loss record or as close as possible. This style of tournament will result in a single winner but all players will play the same number of rounds and have the same number of matches. In his project I also learn about creating and using a PSQL database to hold the tournament players and their results. The python code will interact with that database and using a test file which will simulate the results of a tournament.
